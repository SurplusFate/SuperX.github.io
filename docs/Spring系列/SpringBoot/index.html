                # SpringBoot2 核心技术

                1. 为什么要使用 SpringBoot

                  因为 Spring, SpringMVC 需要使用的大量的配置文件 （xml 文件）

                  还需要配置各种对象, 把使用的对象放入到 spring 容器中才能使用对象

                  需要了解其他框架配置规则.

                2. SpringBoot 就相当于 不需要配置文件的 Spring+SpringMVC

                  常用的框架和第三方库都已经配置好了, 拿来就可以使用了.

                3. SpringBoot 开发效率高, 使用方便多了
