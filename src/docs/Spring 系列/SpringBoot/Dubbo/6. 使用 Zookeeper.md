# 使用 Zookeeper

zookeepeer 客户端依赖：

```xml
<dependency>
    <groupId>org.apache.curator</groupId>
    <artifactId>curator-framework</artifactId>
    <version>5.3.0</version>
</dependency>
```

修改配置文件, 消费者和提供者都要注册.

```xml
  <!--使用 zookeeper -->
<dubbo:registry address="zookeeper://localhost:2181"/>
```
